🚀 SkyTrack – Traffic Management & Smart Redirection Platform

SkyTrack is a full-stack web application designed to help teams efficiently manage, track, and optimize incoming traffic through intelligent link generation and redirection strategies.

Built for call centers, marketing teams, and digital operators, the platform provides powerful tools to monitor user behavior, analyze performance, and control how traffic is routed in real time.

🔗 Live Demo: https://skytrack-web-production.up.railway.app

✨ Key Features
👨‍💼 Agent Dashboard
Secure login system for agents
Generate unique tracking links instantly
Monitor traffic activity and link performance
View real-time analytics for better decision-making
🛠️ Admin Dashboard
Manage agents and system users
Create and control tracking links
Configure redirection rules dynamically
Access system-wide analytics and insights
🔗 Smart Link Generation
Short, shareable URLs with embedded tracking parameters
Track user interactions and traffic sources
Designed for high-volume campaign usage
🔀 Intelligent Redirection
Dynamic landing page handling based on traffic source
Custom redirects (e.g., social media vs direct traffic)
Support for HTTP 301/302 redirection
📊 Traffic Monitoring & Analytics
Detailed logs of link visits and user behavior
Track performance across agents and campaigns
Data-driven insights for optimization
🎨 Responsive UI
Clean and modern interface
Fully responsive design (desktop & mobile)
Consistent white & sky-blue theme
🧠 Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js (Express)
Database: MongoDB Atlas
Deployment: Railway
Version Control: GitHub
